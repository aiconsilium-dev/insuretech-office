# PRD: InsureTech Office (관리사무소 앱)

## 개요

아파트 단지 관리사무소 담당자가 입주민 접수 건을 관리하고,
공용부 사고/비입주민 대리접수를 처리하며, 현장조사를 배정·관리하는 모바일 웹 앱.

## 대상 사용자

- 아파트 관리사무소 직원
- URL 파라미터: `?name=김관리&apt=헬리오시티`

## 핵심 기능

### 1. 대시보드
- KPI: 금일 접수, 처리중, 현장조사 요청
- TYPE별 현황 (A/B/C 카드)
- AI 알림: 현장조사 요청, 긴급 접수
- 최근 접수 리스트

### 2. 접수관리
- TYPE 필터 (전체/A/B/C)
- 출처 필터 (전체/입주민/관리사무소)
- 접수 카드: 위치, 유형, TYPE 배지, 상태 배지
- 상세 확장:
  - TYPE A: 시공사 하자 정보, 하자 상세, 보증기간
  - TYPE B: 면책 사유, 약관 조항
  - TYPE C: AI 적산 금액, 소유자/임차인 부담
- 현장 사진 (3종 플레이스홀더)
- 현장조사 배정 / 상태 변경 버튼
- 4단계 상태 진행바

### 3. 신규접수
**공용부 사고접수**
- 사고유형 선택
- 위치 (동, 층/시설명)
- 사진 3종
- 누수 시 소견서 첨부
- 상세 설명
- AI 분석 → TYPE 결과

**비입주민 대리접수**
- 피해자 정보 (이름, 연락처, 관계)
- 사고 정보 (유형, 위치, 일시, 경위)
- 사진 3종
- 도급업체 관련 토글
- AI 분석 → TYPE C 결과

### 4. 더보기
- **현장조사 관리**: 대기/진행/완료 탭, 현장 출동 배정, 사진 촬영, 소견서 작성
- **품셈 확인**: 2026년 3월 기준 주요 공종 단가
- **구상권 현황**: 도급업체 책임 경합 건 관리
- **설정**: 알림, 계정, 앱 정보

## 데이터 (목업)

8건의 접수 데이터:
- 입주민 접수 5건 (누수, 파손, 결로, 도장, 창호)
- 관리사무소 접수 3건 (배관, 사고, 타일)
- TYPE 분포: A 1건, B 2건, C 5건

## 디자인 토큰

- Primary: `#FF6B35` (orange)
- Text: `#111` (black)
- Background: `#f9fafb` (gray-50)
- Border radius: 14px (카드), 50px (버튼)
- Max width: 430px (모바일 최적화)
- Status badges: received(amber), review(blue), assess(pink), complete(green)
